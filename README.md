# 🍕 Pizza Order App — Vanilla JS Ordering System

Pizza Order App is a fully interactive web application built as the final project for my **WEB 200 class**, where we focused on learning JavaScript fundamentals. The app allows users to order pizzas online, customize toppings, and manage their cart, demonstrating practical skills in vanilla JavaScript, HTML, and SCSS.

---

## 🌐 Live Demo
👉 https://jonhson031.github.io/web200/final__project/index.html

---

## 🚀 Features
- Browse and select pizzas with multiple customization options  
- Add customized pizzas to the cart  
- View and manage cart contents in a modal popup  
- Cart persists across sessions using `localStorage`  
- Checkout form with basic validation  
- Generate a receipt with order summary and customer information  
- Clean, responsive, and user-friendly interface  

---

## 🛠️ Technologies Used
- HTML5  
- SCSS  
- JavaScript (Vanilla JS, ES6+)  
- LocalStorage for persistent cart data  
- DOM manipulation and event handling  

---

## 🧠 Key Concepts Demonstrated
- Dynamic DOM updates based on user actions  
- Managing application state with JavaScript objects and arrays  
- Form validation and submission handling  
- LocalStorage for saving cart data across sessions  
- Modal implementation for cart overview  
- Generating dynamic content (order summary/receipt)  

---

## ⚙️ How It Works
1. Users select pizzas and customize toppings  
2. Customized pizzas are added to the cart and saved in `localStorage`  
3. Cart modal displays all items with quantity and price  
4. Users complete a checkout form with basic validation  
5. On successful validation, the order is placed and a receipt is generated showing the order summary and customer information
